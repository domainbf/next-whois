接口信息

请求地址
bash
GET https://yisi.yun/api/lookup
请求参数
querystring, 必填 - 输入域名进行测试
认证方式

通过请求头或查询参数传递 API Key

bash
# 请求头方式
curl "https://yisi.yun/api/lookup?query=baidu.com" \
  -H "x-api-key: YOUR_API_KEY"

# 查询参数方式
curl "https://yisi.yun/api/lookup?query=baidu.com&apiKey=YOUR_API_KEY"
速率限制

默认限制：每分钟 10 次，每小时 30 次
API Key 限制：每分钟 20 次，每小时 50 次，每天 300 次
错误码参考

错误码	HTTP 状态码	说明
INVALID_API_KEY	401	API Key 无效
API_KEY_DISABLED	401	API Key 已禁用
RATE_LIMIT_EXCEEDED	429	超过频率限制
IP_NOT_ALLOWED	403	IP 不在白名单
